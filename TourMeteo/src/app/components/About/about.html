TourMeteo — une application de comparaison météo entre plusieurs villes, heure par heure

Version: 1.0.0 — Réalisé avec Angular et Tailwind.

Lien du répertoire GitHub : https://github.com/aZor9/TourMeteo

APIs utilisées

  • Géocodage : Nominatim (OpenStreetMap) — recherche d'emplacements
  • Météo : Open-Meteo — données météo horaires (température, vent, weathercode)

Remarques

Les données sont obtenues en temps réel depuis les APIs publiques listées ci‑dessus.

Aucune donnée utilisateur n'est envoyée à des services autres que ceux mentionnés.
